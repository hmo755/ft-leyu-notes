# ft-leyu-notes

## 简介

`ft-leyu-notes` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面可能包含笔记、工具、演示或其他静态内容，彼此之间没有强关联，也不针对任何特定网站或服务。

本仓库旨在提供一个简洁、可访问的静态页面集合，方便查阅和分享。

## 目录说明

仓库根目录下包含多个 HTML 文件，每个文件对应一个独立的页面。部分页面可能引用同目录下的资源文件（如 CSS、JavaScript、图片等）。

- `index.html`：仓库的入口页面，可能包含页面列表或导航。
- `*.html`：其他独立页面文件，命名通常反映其内容或用途。
- `assets/`（可选）：存放公共资源文件，如图片、样式表或脚本。
- `README.md`：本说明文件。

## 页面归档说明

- 每个 HTML 页面均为自包含或依赖仓库内资源，可直接在浏览器中打开使用。
- 页面内容可能随时间更新或新增，历史版本可通过 Git 提交记录查看。
- 仓库不保证所有页面均保持最新或完全兼容所有浏览器环境。

## 维护说明

- 本仓库由个人维护，主要用于归档和分享。
- 欢迎通过 Issues 提交问题或建议，但不承诺及时响应。
- 不接受未经讨论的 Pull Request，除非涉及明显错误或安全修复。
- 仓库内容遵循 MIT 许可证，详情请参见 LICENSE 文件。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或任意 `.html` 文件。
3. 也可通过 GitHub Pages 等服务部署，获取在线访问链接（需自行配置）。
